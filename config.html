Xorg

The Xorg user session is started by the file .xsession. This script does the following things:

  • if not already running, start the keychain program to manage pgp and ssh keys
  • define several useful environment variables
  • load the key bindings in the .Xmodmap file
  • invoke the script to start the windows manager

The file .Xmodmap contains XOrg key bindings. Most of the keys are automatically recognized by the system. SO the only thing the script does is to remap the windows key (between \Ctrl\ and \Alt\ keys) to generate screen escape character.

The file .Xresources contains configuration options of the X11 applications.
